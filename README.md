## How to use

First, "git clone" to copy this repository to your dev machine and then install RubyGems with following command.

```
$ bundle install --without production
```
Migrate DB.
```
$ rails db:migrate
```
Conduct test.
```
$ rails test
```
Launch rails server.
```
$ rails server
```
